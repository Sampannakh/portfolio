Adding Your Portfolio Project Images

Prepare your portfolio images

Select three high-quality images that represent your photography work:

  • For the Portrait Series: A striking portrait photo
  • For the Brand Campaign: A commercial/brand-related photo or video still
  • For the Cinematic Short Film: A cinematic still or film scene

Tip: For optimal display, resize your images to 800x600 pixels and compress them for the web.

Rename your files (optional but recommended)

Rename your images to make them easier to identify:

  • portrait-series.jpg - For your portrait photography
  • brand-campaign.jpg - For your commercial work
  • short-film.jpg - For your cinematic work

Add images to the portfolio folder

Copy your three portfolio images to the assets/images/portfolio/ folder.

You can do this by:

  • Using the file explorer/finder to navigate to your project folder
  • Go to assets → images → portfolio
  • Copy your three images into this folder

Update your HTML file

Open the index.html file and update the three project cards to use your actual images:

Find these sections (around lines 191-280) and change the image sources:

<div class="project-card" data-category="photo"> <div class="project-image"> <img src="assets/images/portfolio/portrait-series.jpg" alt="Portrait Photography" /> <!-- rest of the project card code --> </div> </div> <div class="project-card" data-category="video"> <div class="project-image"> <img src="assets/images/portfolio/brand-campaign.jpg" alt="Video Campaign" /> <!-- rest of the project card code --> </div> </div> <div class="project-card" data-category="cinema"> <div class="project-image"> <img src="assets/images/portfolio/short-film.jpg" alt="Short Film" /> <!-- rest of the project card code --> </div> </div>

Troubleshooting

If your images don't appear:

  • Check the file path - Make sure the path in HTML matches exactly where you placed the files
  • Check file names - File names are case-sensitive, so make sure the case matches exactly (e.g., "Portrait-Series.jpg" is not the same as "portrait-series.jpg")
  • Check file extensions - Make sure you're using the correct extension (.jpg, .jpeg, .png)
  • Browser cache - Try hard-refreshing your browser (Ctrl+F5 or Cmd+Shift+R)
