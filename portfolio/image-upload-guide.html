How to Add Images to Your Photography Portfolio

Step 1: Prepare Your Image Files

Before adding images to your portfolio, make sure they are properly prepared:

  • Use high-quality but optimized images (compressed for web)
  • Maintain consistent aspect ratios for similar image types
  • Name your files descriptively (e.g., portrait-session-2023.jpg rather than IMG12345.jpg)
Pro Tip: Use online tools like TinyPNG or Squoosh to compress your images without losing visible quality.

Step 2: Organize Your Images in the Correct Folders

Your portfolio website uses this folder structure:

portfolio/
├── assets/
│   ├── images/            # Main images folder
│   │   ├── portfolio/     # Portfolio project images
│   │   ├── profile.jpg    # Your profile/about image
│   │   ├── hero-background.jpg  # Hero section background
│   │   └── contact-background.jpg  # Contact section background
├── index.html
├── style.css
└── script.js

Make sure to place your images in the appropriate folders based on their purpose.

Step 3: Update HTML to Include Your Images

To add a profile image to the About section:

<div class="about-image-container">
  <img
    src="assets/images/profile.jpg"
    alt="Sampanna Khanal - Photographer portrait"
  />
</div>

To add a portfolio project image:

<div class="project-card" data-category="photo">
  <div class="project-image">
    <img
      src="assets/images/portfolio/portrait-series.jpg"
      alt="Portrait Photography"
    />
    <!-- rest of the project card code -->
  </div>
</div>

Step 4: Update CSS for Background Images

To change the hero background image, update the CSS in style.css: 

.hero {
  background: url('assets/images/hero-background.jpg') no-repeat center center;
  background-size: cover;
}

To change the contact section background:

.contact::before {
  background: url('assets/images/contact-background.jpg') no-repeat center center;
  background-size: cover;
}

Recommended Image Sizes

  • Hero Background: 1920 × 1080 pixels (16:9 ratio)
  • About/Profile Image: 800 × 1000 pixels (4:5 ratio)
  • Portfolio Project Images: 800 × 600 pixels (4:3 ratio)
  • Contact Background: 1920 × 1080 pixels (16:9 ratio)
